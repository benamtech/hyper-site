## 4. Implementation truth retained

```text
Stage-1 agent proposal
-> provenance/materiality/safety/reviewer compiler
-> TF-IDF/BM25 baseline
-> typed sparse graph + hard constraints
-> bounded observed/graph conjunction expansion
-> sparse concave coverage selection
-> HRR structural encoding
-> one Stage-2 PageConceptJob per region
-> external deterministic validation and repair findings
-> CandidatePageSeed into existing coordinate/job/PageIR/UI path
```

The code remains a graph-and-sparse-set-led compiler with HRR as a first-class structural representation. It is not an HRR model that autonomously discovers evidence or writes pages.
