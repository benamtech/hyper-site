## Decision

The UI framework is now browser-first and page-matrix-native. Agents do not invent pages from a visual prompt. They compile matrix coordinates, vector prototypes, semantic modules, evidence, and approved variants into one of five deterministic layout archetypes under the AMTECH design authority.

## Browser baseline

Tier 2: Chrome/Edge 111+, Firefox 113+, Safari/iOS Safari 16.4+. Every CSS generation pass records 57 feature decisions. No dark mode.

## Canonical rendering

Complete static HTML and CSS remain the product. Canonical JavaScript is zero. Optional interactions cannot change meaning, evidence, links, price, CTA, robots, or publication state.

## 3D boundary

Three.js/R3F is an optional geometry projection. It is poster-first, user-activated, one-scene, demand-rendered, adaptive-DPR, instanced/batched, no-postprocessing by default, and non-canonical. Wasm remains benchmark-only.
